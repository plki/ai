# 智能桌面助手 (AI Desktop Assistant) — 使用说明

> 版本 0.3.0 | Windows 平台 | Python 3.8+

---

## 📦 快速安装

### 方式一：一键安装（推荐）

**双击 `install.bat`**，脚本会自动完成：
1. ✅ 检测 Python 环境
2. ✅ 安装所需的 Python 依赖包
3. ✅ 检测并自动安装 Ollama（本地 AI 引擎）
4. ✅ 引导下载 AI 模型（推荐 `qwen2.5:0.5b` 轻量中文模型）

### 方式二：手动安装

```bash
# 1. 安装 Python 依赖
pip install -r requirements.txt

# 2. 安装 Ollama（如未安装）
#    访问 https://ollama.com/download 下载安装

# 3. 启动 Ollama
#    Ollama 安装后会自动开机自启，也可以手动打开

# 4. 下载 AI 模型（选一个）
ollama pull qwen2.5:0.5b    # 397MB，轻量中文，速度最快
ollama pull qwen2.5:1.5b    # 1.1GB，中文效果更好（推荐）
ollama pull tinyllama       # 637MB，通用英文

# 5. 启动助手
python main.py
```

## 🚀 启动方式

| 方式 | 操作 | 说明 |
|------|------|------|
| **一键启动** | 双击 `launcher.bat` | 自动管理 Ollama 生命周期（启动→运行→关闭），推荐 |
| **直接启动** | `python main.py` | 需确保 Ollama 已在运行 |
| **快捷工具** | 双击 `pull_qwen.py` | 单独下载 qwen2.5:0.5b 中文模型 |

> 💡 **launcher.bat 的优势**：自动检测 Ollama 是否运行，若未运行则自动启动；退出助手后自动关闭自己启动的 Ollama，不干扰用户自己运行的 Ollama。

## 🎮 命令大全

启动后进入 CLI 交互界面，支持以下命令：

### 基本命令

| 命令 | 说明 |
|------|------|
| `help` 或 `h` | 显示帮助信息 |
| `exit` 或 `q` | 退出程序 |
| `clear` 或 `cls` | 清屏 |
| `banner` | 重新显示启动横幅 |

### 💬 AI 对话

| 命令 | 说明 |
|------|------|
| `chat` | 进入连续对话模式（输入 `exit` / `back` 退出） |
| `chat ask <问题>` | 快速问答，一次性返回结果 |
| `chat models` | 查看当前可用的 AI 模型 |
| `chat switch <模型名>` | 切换 AI 模型 |
| `chat clear` | 清空对话历史 |

> **在对话模式中**，AI 可以自动调用工具执行操作（文件管理、网页搜索、系统信息等），无需手动切换命令。

### 📁 文件管理

| 命令 | 说明 |
|------|------|
| `file list [路径]` | 列出目录内容（默认当前目录） |
| `file find <关键词> [路径]` | 搜索文件（默认搜索当前目录） |
| `file sort [路径]` | 按类型整理文件（图片/文档/视频/代码等自动归类） |

> 文件整理会创建分类子目录（如"图片""文档""压缩包"等），将文件自动移入对应目录。

### 🔍 网页工具

| 命令 | 说明 |
|------|------|
| `web fetch <URL>` | 获取网页内容，显示标题、链接数和下载资源 |
| `web search <关键词>` | 搜索互联网信息（使用 DuckDuckGo） |
| `web download <URL>` | 下载网络文件到 `data/downloads/` |

### 📋 自动化任务

| 命令 | 说明 |
|------|------|
| `task list` | 列出所有可用任务 |
| `task run <名称或编号>` | 运行指定任务 |
| `task add <名称> <描述> <命令>` | 添加自定义任务 |

**内置任务一览：**

| 任务 | 说明 |
|------|------|
| 系统信息 | 查看操作系统、CPU、内存使用情况 |
| 磁盘分析 | 查看各磁盘分区容量和使用情况 |
| 清理临时文件 | 删除 Windows 临时文件释放空间 |
| 网络诊断 | Ping 测试网络连通性 |
| 进程列表 | 按内存占用排序显示 Top 10 进程 |
| 关机定时 | 设置 30 分钟后自动关机 |
| 取消关机 | 取消已设置的定时关机 |
| WiFi 密码查看 | 查看当前连接的 WiFi 密码 |

### 🤖 模型管理

| 命令 | 说明 |
|------|------|
| `model list` | 列出已下载的模型 |
| `model search [关键词]` | 查看/搜索推荐模型列表 |
| `model download <名称或编号>` | 从 HuggingFace 镜像下载模型 |
| `model delete <名称>` | 删除已下载的模型 |

**推荐模型：**
1. **Qwen2.5-0.5B-Instruct** (~350MB) — 阿里千问，轻量中文强
2. **Llama-3.2-1B-Instruct-Q4** (~700MB) — Meta，英文强
3. **TinyLlama-1.1B-Chat** (~700MB) — 均衡通用
4. **DeepSeek-R1-Distill-Qwen-1.5B** (~1GB) — 推理能力强

### 📦 备份管理

| 命令 | 说明 |
|------|------|
| `backup list` | 列出所有备份记录 |
| `backup run <路径> [名称]` | 备份指定目录到 `data/backups/` |
| `backup restore <编号> [目标路径]` | 恢复指定备份 |

### ⏰ 定时任务

| 命令 | 说明 |
|------|------|
| `schedule list` | 列出所有定时任务 |
| `schedule add <名称> <类型> <配置> <周期>` | 添加定时任务 |
| `schedule remove <编号>` | 删除定时任务 |
| `schedule start` | 启动调度器（后台运行） |
| `schedule stop` | 停止调度器 |

> 周期格式：`30m`=每30分钟，`1h`=每小时，`1d`=每天

### 🗣️ 自然语言模式

直接输入自然语言（不以 `chat`/`file`/`web` 等命令词开头），助手会自动理解你的意图并执行。

**示例：**
- "帮我看看 D 盘有哪些文件" → 自动调用 `file list D:/`
- "搜索一下 Python 教程" → 自动调用 `web search Python教程`
- "查看一下系统状态" → 自动调用 `task run 系统信息`
- "帮我把桌面上的文件分类整理" → 自动调用文件整理
- "下载 https://example.com/file.zip" → 自动下载文件

## ❓ 常见问题

### Q: 启动后报错 "Ollama 未运行"？
- 确保已安装 Ollama（访问 [ollama.com/download](https://ollama.com/download)）
- 使用 `launcher.bat` 启动会自动处理
- 或在命令行运行 `ollama serve` 手动启动

### Q: AI 对话没有反应或返回空白？
- 运行 `ollama list` 检查是否有模型
- 运行 `chat models` 查看助手识别的模型
- 若无模型，运行 `ollama pull qwen2.5:0.5b` 下载一个

### Q: 下载模型速度很慢？
- 配置文件的 `huggingface_mirror` 默认使用 `hf-mirror.com`（国内镜像）
- 可以手动修改为其他镜像站

### Q: 如何更换 AI 模型？
- 对话中运行 `chat switch <模型名>` 切换
- 新模型需先通过 `ollama pull <模型名>` 下载

### Q: 文件整理会删除文件吗？
- **不会删除任何文件**，仅将文件按类型移动到对应的分类子目录中
- 可手动移回原位置

### Q: 如何卸载？
- 直接删除 `E:\aiyidai\` 整个目录即可
- 如需卸载 Ollama，在"添加或删除程序"中卸载
